# app-market-analysis
Проект посвящен анализу данных приложений из Google Play Store

Цель проекта - изучить рынок мобильных приложений и выявить
основные закономерности

В ходе анализа были исследованы:
- популярность категорий приложений
- распределение рейтингов
- влияние типа приложения (платное/бесплатное)
- средние рейтинги по категориям

Инструменты:
Python, Pandas, Matplotlib
